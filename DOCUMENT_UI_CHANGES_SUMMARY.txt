WHY THESE CHANGES?
===============================================================================

Previously:
- Only 5 documents per page → hard to find documents
- Documents needed admin approval → delays in access
- Confirm button cluttered interface → confusing for users
- Approve button took space → more clicks needed

Now:
- 50-500 documents per page → easy to find & browse
- Documents publish instantly → immediate access for all staff
- Cleaner interface → less confusing, fewer buttons
- Admin can delete if mistakes → can remove duplicates/errors
